# Atlas do Ensino

Acervo editorial aberto sobre a história da sala de aula, os métodos de ensinar e a pesquisa do ensino.

## Estrutura

- `data/content.mjs`: catálogo editorial;
- `assets/images`: imagens locais e otimizadas;
- `src`: estilos e comportamento da interface;
- `build.mjs`: gerador das páginas estáticas;
- `docs`: saída estática publicada pelo GitHub Pages.

## Desenvolvimento

```sh
npm run build
npm run serve
```

O GitHub Pages publica o diretório `docs` da branch `main`.

## Direitos

Cada imagem possui indicação de fonte e licença em sua página. Os textos editoriais do projeto não adotam automaticamente a licença das obras reproduzidas.
